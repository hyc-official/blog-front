电脑密码被家长改了，怎么才能夺回使用权？

相信大家都有过这样的经历。

下面给出几种方案，可行性与安全性评分用 ⭐ 的数量表示，最多 5 个。

思路#1 钻操作系统空子

即使用操作系统的某些功能达到使用电脑的目的。

方案#1 新用户

  • 可行性：⭐⭐⭐⭐⭐
  • 安全性：⭐⭐

（家长让你用电脑时）在电脑设置或控制面板中新建用户即可。

优点就是使用方便，但是每次使用完后要记得注销并切换登录用户，否则就会被发现的！/kk

如果家长细查，也不是查不到。

方案#2 清除密码

  • 可行性：⭐⭐⭐⭐
  • 安全性：⭐

此方法需要使用者有一定的计算机操作基础。

电脑开机时进入 Windows PE 环境（至于上哪去找就不是我的逝啦），使用某些工具清除用户密码即可。

缺点很明显：密码没了，人也没了。

关于 Windows PE：建议使用微 PE，link。

思路#2 不使用电脑硬盘内的系统

顾名思义，就是使用其他来源的系统，如内置硬盘双系统、外接硬盘等。

方案#3 多系统

  • 可行性：⭐⭐⭐
  • 安全性：⭐⭐⭐⭐

此方法需要使用者有一定的计算机操作基础。

在电脑内置硬盘上安装多个系统，或者在多个内置硬盘上安装多个系统。

安全性较高，但是制作过程较为危险。

由于本方法难度较高，这里给出制作方法。

制作教程

  1. 制作一个系统安装盘（可以从微硬官网下载制作工具，或者下载 iso 镜像并写入安装盘）。
  2. 插着安装盘重启电脑，进入 BIOS 更改第一启动项为安装盘（或在下一步临时使用安装盘启动）。
  3. 退出 BIOS，重启电脑，电脑会从安装盘中启动。
  4. 接下来跟着安装向导安装即可，注意不要安装到已经安装了系统的硬盘分区中，否则原系统就无啦会被覆盖。

方案#4 WTG

  • 可行性：⭐
  • 安全性：⭐⭐⭐⭐⭐（在 WTG 启动盘保存良好的条件下）

WTG，即 Windows To Go 的缩写，是一种可以将 Windows 8 及以上系统装入 U 盘即插即用的技术，了解更多。

由于 Windows To Go 的制作方法十分困难，所以这里给出教程。

制作教程

下为官方工具制作教程（参考文章 link），如果任意一步出现问题，可以使用第三方工具制作教程。

  1. 制作前准备：一个高速 U 盘 / 硬盘（速度一定要快，并且是复制小文件时快）和 Windows 企业或教育版镜像（可以在 link 处下载）。
  2. 打开控制面板（这个应该不用我多说了吧）。
  3. 将右上角的“查看方式”改为“大图标”，找到“Windows To Go”选项并点击进入。
  4. 在方框中选择自己的 U 盘 / 硬盘，点击“下一步”。
  5. 点击“添加搜索位置”，把下载的 Windows 镜像的文件夹添加。
  6. 在上面的方框中选择 Windows 8/8.1/10 企业版/教育版，点击“下一步”。
  7. 选择设置 Bitlocker，点击“下一步”或“跳过”。
  8. 点击“创建”，休息一下吧。
  9. 工具制作完成后，制作工具会询问是否在重启电脑时从 WTG 启动盘启动，可根据自己的情况选择，若选择“否”，则需要在电脑开机时手动调整启动项。
  10. WTG 已就绪，进入 WTG 系统时需要先进行个性化配置，就像新系统刚装好时一样。
