题解 LuoguP9160 multiset

开始之前先说一句：我校大佬出的题我必须做！

前置知识

真子集：若集合 BBB 是集合 AAA 的子集且 A≠BA \ne BA=B，则称 AAA 为 BBB 的真子集。

思路

前置定义

令 ∣X∣|X|∣X∣ 为集合 XXX 的元素个数。

设 S={s1,s2,⋯ ,sn}S = \{s_1, s_2, \cdots, s_n\}S={s1​,s2​,⋯,sn​} 且 s1≤s2≤⋯≤sns_1 \le s_2 \le \cdots \le s_ns1​≤s2​≤⋯≤sn​。

设 T={t1,t1,⋯ ,t∣T∣}T = \{t_1, t_1, \cdots, t_{|T|}\}T={t1​,t1​,⋯,t∣T∣​} 且 t1≤t2≤⋯≤t∣T∣t_1 \le t_2 \le \cdots \le t_{|T|}t1​≤t2​≤⋯≤t∣T∣​。

思路推导

由于题目要求 TTT 是 SSS 的真子集，所以 T≠ST \ne ST=S，于是我们需要去掉至少一个数，但在这里，我们要的是最大的 TTT，所以我们只需要去掉一个数，即 ∣T∣=n−1|T| = n - 1∣T∣=n−1。

首先，我们要知道，直接把 SSS 中的最大值去掉（即 T={s1,s2,⋯ ,sn−1}T = \{s_1, s_2, \cdots, s_{n - 1}\}T={s1​,s2​,⋯,sn−1​}）是符合条件的，但可能不是最小。

我们要使 TTT 中元素的和最大，那么我们去掉的数肯定是越小越好，但是如果我们去掉了一个在 SSS 中只有一个的数（设其为 sis_isi​），就会导致另外一个以它为前驱的数在 SSS 中有前驱，但在 TTT 中没有，是不符合题目条件的，所以我们去掉的数应该是最小的有重复的数。

那如果 SSS 中没有重复的数呢？那我们去掉的数应该不是任何数的前驱，而只有最大的那个数（即 sns_nsn​）符合条件。

总结

显然对于第一问，∣T∣=n−1|T| = n - 1∣T∣=n−1。

第二问有以下两种情况：

  • 若存在整数 i<ji < ji<j 使 si=sjs_i = s_jsi​=sj​，则 ∑a=1∣T∣ta=(∑b=1nsb)−si\sum\limits_{a = 1}^{|T|}t_a = (\sum\limits_{b = 1}^{n}s_b) - s_ia=1∑∣T∣​ta​=(b=1∑n​sb​)−si​。
  • 否则 ∑a=1∣T∣ta=∑b=1n−1sb\sum\limits_{a = 1}^{|T|}t_a = \sum\limits_{b = 1}^{n - 1}s_ba=1∑∣T∣​ta​=b=1∑n−1​sb​.
